# Robôs e Automação

## Conceito
Robôs assistentes recebem tarefas designadas pelo jogador para aumentar produtividade e reduzir erros.

## Tipos de robô
- **Coletor:** busca itens no pátio e leva para triagem.
- **Classificador:** identifica categoria com IA básica.
- **Transportador:** move lotes para lixeira/caçamba.
- **Mantenedor:** reduz downtime e custo de manutenção.

## Sistema de tarefas
- Designar zona + prioridade + categoria-alvo.
- Ex.: “Robô Beta: focar em plástico no Setor B por 3 min.”

## Upgrades de robô
- Velocidade
- Precisão
- Bateria/autonomia
- Capacidade de carga
- Módulos especiais (detector de perigoso)

## Balanceamento
- Robô acelera, mas custa energia/manutenção.
- Dependência excessiva reduz bônus de habilidade manual.
