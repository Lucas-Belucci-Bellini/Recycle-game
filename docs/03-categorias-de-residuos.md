# Categorias de Resíduos e Regras de Triagem

## Categorias base
1. **Papel** (azul)
2. **Plástico** (vermelho)
3. **Vidro** (verde)
4. **Metal** (amarelo)
5. **Orgânico** (marrom)
6. **Rejeitos** (cinza)
7. **Perigosos** (laranja)
8. **Eletrônicos** (preto)

## Exemplo de itens por categoria
- Papel: jornal, caixa de papelão, caderno.
- Plástico: garrafa PET, tampa, embalagem.
- Vidro: pote, garrafa quebrada (com cuidado).
- Metal: lata de alumínio, arame.
- Orgânico: casca de fruta, resto de comida.
- Rejeitos: papel higiênico usado, absorvente.
- Perigosos: pilha, bateria, solvente.
- Eletrônicos: celular antigo, cabo, teclado.

## Regras educacionais
- Item contaminado pode mudar de categoria (ex.: papel engordurado vira rejeito).
- Alguns itens têm múltiplas rotas (ex.: vidro inteiro para reuso, quebrado para reciclagem).

## Modo avançado
- Rotulagem por subtipos (PET, PEAD, alumínio, aço).
- Indicação de destino final (reciclagem, compostagem, aterro, logística reversa).
