# UI/UX e Acessibilidade

## Princípios
- Clareza visual por cor + ícone + texto.
- Feedback instantâneo de acerto/erro.
- Interface limpa para reduzir sobrecarga cognitiva.

## Componentes-chave
- HUD de capacidade de lixeiras/caçambas.
- Painel de tarefas dos robôs.
- Comparador de estruturas.
- Glossário rápido de resíduos.

## Acessibilidade
- Modo daltônico (paleta adaptada).
- Narração de feedback essencial.
- Tamanho de fonte ajustável.
- Controles por teclado e mouse/toque.

## Onboarding
- Tutorial em 3 fases com prática assistida.
- Dicas contextuais que desaparecem com domínio.
