# Recycle-game

Jogo web de reciclagem separado em 3 camadas:
- `index.html` (estrutura)
- `styles.css` (estilo)
- `game.js` (lógica)

## Sistemas adicionados
- Desafios progressivos
- Níveis por pontuação
- Recompensas por nível
- Checkpoint automático a cada 100 pontos

## Unreal Engine
Também dá para evoluir este protótipo para Unreal Engine (Blueprints/C++), mas nesta entrega deixei tudo funcional no navegador para validar gameplay primeiro.
